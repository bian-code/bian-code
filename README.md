## Featured Projects

- **CaviteGo** — CaviteGO is a mobile-based public transit navigation and routing prototype system designed to streamline commuting within Cavite.

The system focuses on practical, real-world utility for commuters. It maps out public transit paths, organizes local route data, and provides users with a reliable, step-by-step routing system to navigate the area. Essentially, it takes the guesswork out of local travel by turning informal transit networks into a clear, structured digital guide for everyday commuting.

- **Paradigma Work Order System** — The Work Order System is a centralized backend application built with Spring Boot that digitizes and automates the lifecycle of service and maintenance requests.

It exposes a set of secure RESTful APIs to handle the entire workflow: ingesting new service requests, dynamically routing tasks to technicians, tracking real-time status updates (e.g., Pending to In Progress), and logging final resolutions. By managing these state transitions and persisting the data to a relational database, the system replaces manual paperwork with a reliable, structured digital pipeline.
